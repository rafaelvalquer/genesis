# Genesis Defense

Jogo de defesa tática por ondas, extraído do mini-game de **Colônia Gênesis** e transformado em uma campanha independente totalmente front-end.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Validação

```bash
npm run test:unit
npm run build
```

O progresso e as configurações são armazenados apenas no `localStorage`. O build em `dist/` pode ser publicado em qualquer hospedagem estática.
